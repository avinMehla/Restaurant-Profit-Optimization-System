# Restaurant Profit Optimization System

A full-stack analytics platform designed to help restaurant businesses improve profitability through data-driven insights and recommendations.

## Tech Stack

### Frontend

* React
* React Router
* Axios
* Recharts

### Backend

* Spring Boot
* Spring Data JPA
* Hibernate

### Database

* PostgreSQL

### Analytics (In Progress)

* Power BI

---

## Features Implemented

### Dashboard

* Revenue, Profit, Margin, Quantity Sold, and Average Selling Price KPIs
* Top Revenue Item
* Most Profitable Item
* Highest Margin Item
* Top 5 Most Profitable Menu Items
* Revenue vs Profit Visualization

### Menu Performance

* Menu Item Performance Analysis
* Search Functionality
* Sorting by Revenue, Profit, and Margin
* Item Ranking

### Recommendations

* Profit Optimization Recommendations
* Recommendation Filtering
* Recommendation Statistics Dashboard

---

## Architecture

```text
React
  ↓
Axios
  ↓
Spring Boot REST APIs
  ↓
Service Layer
  ↓
JPA Repository
  ↓
PostgreSQL Views
```
